# Plant Stress and Regulatory Review Storyline

Use the progressive argument:

`Why -> What -> Where -> How -> Network -> Integration -> Application -> Future`

Translate it into:

`field pressure -> focal object -> stress involvement -> causal mechanisms -> signalling crosstalk -> evidence-bounded unified model -> breeding value -> decisive future experiments`

Do not use this as a rigid table of contents. Merge, split, or omit stages according to the evidence.

## Required Questions

- Why does this topic matter now, and what synthesis gap exists?
- What exactly is the focal gene family, pathway, hormone, regulator, or mechanism?
- In which stress, tissue, developmental, species, and environmental contexts is it implicated?
- Which upstream inputs, focal-object controls, direct targets, interactions, and feedback loops are causally supported?
- Which hormone and signalling systems connect to it, and are the relations synergistic, antagonistic, sequential, or context dependent?
- Which conserved core and context-specific branches explain the evidence and apparent contradictions?
- Which alleles, editing strategies, network interventions, or breeding routes are realistic, and what are their risks?
- Which experiments would discriminate between competing models?

## Evidence Ladder

1. Expression or co-expression: candidate involvement.
2. Genetic perturbation with phenotype: functional contribution.
3. Complementation or allele comparison: stronger causal support.
4. Binding, biochemical, or interaction assay: molecular link.
5. Genetic and molecular evidence in the same system: pathway placement.
6. Multi-environment or field validation: translational relevance.

Reconcile conflicts through genotype, tissue, stage, dose, duration, assay, and environment. Distinguish demonstrated links, indirect associations, and hypotheses in both prose and figures.

## Unified Model

When evidence permits, model:

`stress inputs -> stress-specific sensors -> shared signalling nodes -> focal object or complex -> direct and indirect targets -> cellular and physiological outputs -> growth or yield consequences`

Show conserved relations, stress-specific branches, feedback, context modifiers, intervention points, and missing causal edges. Do not call an object a hub based on publication count.
