# Plant Stress Review Storyline

Use this reference to design narrative reviews centred on a plant gene family, transcription factor, signalling pathway, hormone, ncRNA, epigenetic mechanism, molecular regulator, or comparable focal object in stress biology.

## Contents

1. Core storyline
2. Outline construction
3. Section logic
4. Crosstalk and unified model
5. Evidence architecture
6. Figures and tables
7. Adaptation rules
8. Quality gate

## Core Storyline

Build the review as a progressive argument:

`Why -> What -> Where -> How -> Network -> Integration -> Application -> Future`

Translate this into:

`field pressure -> focal object -> stress involvement -> causal mechanisms -> signalling crosstalk -> unified regulatory model -> breeding value -> unresolved questions`

The focal object is not the story by itself. The story is how the object helps explain plant adaptation across biological scales and how that knowledge can be tested or translated.

Do not use this chain as a rigid table of contents. Merge stages when the evidence base is small; split them when distinct mechanisms or stress classes support separate synthesis. Omit unsupported categories rather than padding them with weak expression studies.

## Outline Construction

Before drafting, create an argument map with these fields:

| Stage | Required question | Expected output |
|---|---|---|
| Why | Why does this topic matter now? | Field pressure, crop consequence, synthesis gap, review thesis |
| What | What exactly is the focal object? | Scope, origin/classification, structure, localization, normal function |
| Where | In which stress contexts is it implicated? | Abiotic and biotic evidence map, with exclusions |
| How | Through which causal mechanisms does it act? | Upstream inputs, downstream targets, interactions, feedback, regulation |
| Network | Which hormone and signalling systems connect to it? | Shared nodes, synergy, antagonism, tissue or stage dependence |
| Integration | What general model explains the evidence? | Conserved core, context-specific branches, trade-offs, uncertainty |
| Application | What can realistically be engineered or selected? | Targets, alleles, editing, pyramiding, omics-assisted routes, risks |
| Future | Which experiments would change the field? | Specific gaps, combined stresses, spatial resolution, field validation |

For every section, state:

`section claim -> evidence groups -> contradiction or boundary -> interpretation -> transition to next section`

## Section Logic

### 1. Introduction: Why

Use:

`environmental change or production constraint -> plant stress burden -> need to understand adaptation -> candidate regulatory layer -> unresolved synthesis gap -> review scope and thesis`

Avoid generic climate-change openings that never connect to the focal object. State what this review integrates that existing reviews or fragmented primary studies do not.

### 2. Focal Object: What

Define only the background needed to understand stress function:

- origin, evolution, and classification
- conserved domains, motifs, localization, and molecular properties
- normal roles in growth, development, reproduction, or metabolism
- plausible bridge from normal function to stress adaptation

Do not let taxonomy dominate the review. End by explaining why the object's baseline biology predicts a stress-related role.

### 3. Stress Landscape: Where

Organize first by reader-facing stress context when this improves accessibility:

- abiotic: drought, salinity, heat, cold, flooding, nutrient limitation, heavy metals, oxidative stress, or other supported stresses
- biotic: fungi, bacteria, viruses, oomycetes, nematodes, herbivores, or other supported challenges

Within each stress subsection, synthesize:

`stress dimension -> focal-object response -> strongest functional evidence -> adaptive output -> unresolved mechanism`

Do not force equal coverage. Allocate space according to evidence strength, conceptual importance, and relevance to the review thesis.

### 4. Mechanism: How

Reorganize observations into causal layers:

1. Upstream control: perception, Ca2+, ROS, MAPK, kinases, transcriptional induction, chromatin state, or other supported inputs.
2. Focal-object regulation: abundance, localization, complex formation, degradation, phosphorylation, ubiquitination, or other modifications.
3. Downstream action: direct targets, transporters, enzymes, defence genes, developmental genes, or physiological programmes.
4. Feedback and feed-forward loops.
5. Outputs: homeostasis, cellular protection, immunity, growth adjustment, reproduction, survival, or yield.

Distinguish direct evidence from inferred pathway placement. Expression change alone does not establish upstream or downstream causality.

### 5. Hormone and Signalling Network

Map relevant hormone and signalling connections, such as ABA, JA, SA, ethylene, auxin, gibberellin, brassinosteroid, cytokinin, strigolactone, ROS, Ca2+, and MAPK modules.

For each connection, classify the relationship:

- upstream induction or repression
- downstream target or output
- synergistic or antagonistic crosstalk
- feedback regulation
- tissue-, stage-, species-, or stress-dependent switching

Avoid a hormone-by-hormone list without explaining network consequences.

### 6. Crosstalk and Integration

This is the main synthesis layer. Resolve three interfaces when supported:

1. Stress-stress: shared and distinct nodes across drought, salinity, temperature, nutrient, pathogen, and herbivory responses.
2. Abiotic-biotic: how one stress state changes immunity or how defence changes environmental tolerance.
3. Development-stress: growth, flowering, senescence, reproduction, yield, and tolerance trade-offs.

Ask:

- Which nodes are reused across contexts?
- Which effects reverse by tissue, developmental stage, species, stress intensity, or timing?
- Which apparent contradictions arise from experimental design rather than biology?
- Does the focal object behave as a hub, context-dependent switch, downstream effector, or one component of a distributed network?

Use `hub` only when multiple independently supported inputs and outputs converge. Do not infer centrality from publication count.

## Crosstalk and Unified Model

End the synthesis with an evidence-bounded model:

`stress inputs -> stress-specific sensors -> shared signalling nodes -> focal object or complex -> direct and indirect targets -> cellular and physiological outputs -> growth/yield consequences`

The model must identify:

- conserved core relationships
- stress-specific branches
- positive and negative feedback
- tissue, developmental, species, and environmental modifiers
- demonstrated links versus proposed links
- major missing causal edges

The model figure and prose must make the same claims. Introduce the model before the conclusion; do not hide the review's main intellectual contribution in a final summary paragraph.

## Evidence Architecture

Create a literature matrix with at least:

`species | gene/pathway | stress | tissue/stage | perturbation | molecular evidence | physiological evidence | field evidence | direct target | direction of effect | support grade | limitation`

Use an evidence ladder:

1. Expression or co-expression: candidate involvement.
2. Genetic perturbation with phenotype: functional contribution.
3. Complementation or allele comparison: stronger causal support.
4. Binding, biochemical, or interaction assay: molecular link.
5. Genetic plus molecular evidence in the same model: pathway placement.
6. Multi-environment or field validation: translational relevance.

Build sections around evidence groups and explanatory questions, not species catalogues. Explicitly reconcile conflicting results through genotype, tissue, developmental stage, stress dose, duration, assay, and environment.

## Figures and Tables

For a full review, propose:

1. A scope figure: focal object and major biological domains.
2. An evidence table: stress, species, perturbation, mechanism, phenotype, and support level.
3. A mechanism/network figure: upstream signals, object regulation, targets, and outputs.
4. A unified model figure: common core, context-specific branches, crosstalk, trade-offs, and intervention points.
5. A future-directions box: gap paired with decisive experiment.

Use visual encoding to distinguish activation, repression, physical interaction, indirect association, and hypothesis. Do not draw a causal arrow when the evidence supports only correlation.

## Application and Future Directions

Evaluate:

- elite alleles and natural variation
- CRISPR editing, promoter editing, or tissue-specific perturbation
- gene pyramiding and network-aware engineering
- GWAS, transcriptomics, proteomics, epigenomics, single-cell, and spatial omics
- growth penalties, pleiotropy, genotype dependence, combined-stress effects, and field stability

Frame future directions as testable programmes:

`knowledge gap -> why it matters -> decisive method or experiment -> expected discriminating result`

Prioritize combined and sequential stresses, cell-type and spatial resolution, temporal dynamics, causal perturbation, diverse germplasm, and field validation when relevant. Mention AI-assisted breeding only when connected to defined data, prediction, validation, and deployment steps.

## Adaptation Rules

- If the topic concerns only abiotic or only biotic stress, keep the missing domain as an explicit scope exclusion.
- If mechanistic evidence is sparse, use an evidence-gap review and avoid pretending a mature network exists.
- If the focal object is a technology or resource, use the technology archetype in `plant-review-writing.md` instead.
- If the review is systematic or scoping, preserve this narrative synthesis but also follow `systematic-review-and-search.md`.
- If development is central, place normal developmental function before stress roles and return to the development-stress trade-off in integration.
- If breeding is central, introduce agronomic constraints early and require field or multi-environment evidence before strong application claims.

## Quality Gate

Before approving the outline or draft, verify:

- The review has a one-sentence thesis, not just a topic.
- The sequence moves from phenomenon to mechanism to integration to application.
- Abiotic and biotic coverage reflects the declared scope and available evidence.
- Mechanism sections identify upstream inputs, focal regulation, downstream targets, and outputs.
- Hormone discussion explains relationships rather than listing pathways.
- Crosstalk reconciles contexts, contradictions, and growth-stress trade-offs.
- A unified model adds an interpretation not already obvious from the section summaries.
- Application claims match causal and field evidence.
- Future directions specify decisive experiments.
- The outline contains proposed figures, evidence tables, and explicit citation slots.
