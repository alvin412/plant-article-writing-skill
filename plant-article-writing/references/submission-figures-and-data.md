# Submission, Figures, and Data

## Contents

1. Figure and table contract
2. Submission statements
3. Data and code availability
4. Final submission audit

## 1. Figure and Table Contract

For every major figure define:

- one-sentence conclusion
- unique evidence supplied by each panel
- controls and comparisons
- sample size and replicate type
- statistical test, correction, error-bar definition, and exact significance notation
- source-data location
- image-integrity and processing notes
- relationship to manuscript claims

Drop decorative or redundant panels. Keep panel labels, legends, units, gene/protein typography, colours, and category order consistent.

Audit:

- figure and table callouts
- numbering and order
- captions and abbreviations
- readable labels
- no overlap, clipping, or misleading axis
- no unsupported visual inference
- source data and image provenance

Use a dedicated publication-figure skill when drawing or rebuilding figures.

## 2. Submission Statements

Include when applicable:

- Data Availability
- Code Availability
- Materials or Protocol Availability
- Author Contributions using CRediT
- Funding
- Competing Interests
- Ethics statement
- Acknowledgements
- AI assistance disclosure
- Supplementary information description

Follow the current target-journal instructions when they differ.

## 3. Data and Code Availability

Inventory:

- raw and processed data
- source data for figures and tables
- sequencing accessions
- image data
- analysis scripts and software environment
- models and parameter files
- reused public datasets
- restricted or third-party data

Map each item to a repository, accession/DOI, supplement, controlled-access route, or justified restriction. Do not invent identifiers or licences.

Flag unsupported `available upon request` wording unless a real legal, ethical, commercial, or third-party constraint exists.

## 4. Final Submission Audit

Check:

- target-journal article type and length
- title, abstract, keywords, main text, references, figures, tables, supplement
- line numbering or anonymization if required
- reference and figure style
- journal-specific section order, citation style, bibliography export, and manuscript template
- declarations
- reporting checklist
- file names and version
- English/Chinese alignment for the user's bilingual record

Record unresolved journal-specific items as `WARN` or `BLOCK`.

When requested, create LaTeX/PDF or reference-manager sidecars only after the verified Word package is complete. These exports must preserve the same manuscript version and citation set.
